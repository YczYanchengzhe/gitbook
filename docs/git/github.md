# 一.搜索使用

#### 如何使用GitHub快速搜索到想要的开源项目?

在搜索框不输入 , 直接确认 , 进入高级搜索界面,在这里就可已按照需求进行选择性搜索拉!

```shell
# 搜索 包含文件名为 application.property 的项目 , 并且该文件中包含spring字段
'spring' filename:application.property
```
